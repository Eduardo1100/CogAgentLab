From a team cognitive architecture to an agent-directed research lab.

Continuity of Mind began as a collaborative Global-Workspace-style architecture for LLM agents. CogAgentLab is the subsequent system for running it across environments, preserving evidence, diagnosing failures, and handing focused changes to coding agents for human review.

Current system

The research contribution is the improvement loop.

CogAgentLab connects a cognitive multi-agent runtime to environment adapters, persistent experiment records, analyst traces, and a local coding-agent workflow. The human remains responsible for deciding which changes survive.

  1. RunExecute a bounded debug episode.
  2. ObservePersist metrics, transcripts, and an analyst trace.
  3. DiagnoseTurn the latest failure evidence into a focused prompt.
  4. ChangeClaude Code or Codex implements and checks one iteration.
  5. ReviewEduardo reviews, keeps, revises, or rejects the branch.

Attribution and limitations

A strong link should show where the evidence stops.

Collaborative origin

Historical CognitiveLLM git history includes Eduardo Cortes, Bhaskar Mishra, Yixiao Wang, Will Cai, and Aaron Zheng. The core historical gwt_agent.py work specifically includes Eduardo, Bhaskar, and Yixiao. Eduardo's later work integrated and extensively redesigned that architecture, ran the evaluation, and developed CogAgentLab.

Reproduction boundary

The historical run used uncommitted code later captured in a same-day commit, so the exact dirty diff is unavailable. The result is auditable from metrics and traces, but not represented as a clean one-command historical reproduction.

Publication boundary

The showcase code and original authored content have narrowly scoped licenses. Those notices do not relicense the existing CogAgentLab codebase, source exports, model outputs, or historical team material. Award evidence and sponsor naming remain outside this page.
